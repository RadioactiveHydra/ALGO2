Testing

The project is tested against the unittest.c file. The file tests the math and ADTs files, and more tests may be added later. It should also check compiling of the entire project on a linux machine.

  • On Travis CI
  • On Coverity

The programs are tested on Linux machines and a script is included to install the dependencies

The code will work on MAC OSX but you will need to manually install the dependencies

Parts of the project will not compile on Windows since some libraries do not support Windows (NCURSES...). A Makefile for Windows will be added to compile all the code compatible with Windows
