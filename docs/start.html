Getting Started

  • $git clone https://github.com/Team-ALGO2/ALGO2.git - Clone the code onto your machine
  • $cd c-calculator - Change your directory to be the one you just downloaded
  • $make - Make the binaries from the code
  • Additional code for webserver
  • $cd Server - Change your directory to be the one of the server
  • $make - Make the binaries from the code
  • $./server - Start the server
