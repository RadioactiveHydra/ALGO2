11 ### Calculator:
12 - Boot: When booting, you can select multiple modes using ./calc MODE. The modes can be:
13  - **DEC** for decimal(default)
14  - **BIN** for binairy
15  - **HEX** for hexadecimal
16  - **AST** for astronomical.
17 - Syntax: When using the calculator, there is syntax that you need to folow so that your equastions will work properly:
18  - **NOTE: Make sure to seperate every part of your equasion using space.**
19  - For a number, just spell them out using **0-9** in decimal mode, **0-1** in binairy and from **0-F** in hexadecimal.
20  - For normal equasions please use these signs:
21  - "+" for addition
22  - "-" for substraction
23  - "*" for multiplication
24  - "/" for division
25  - "%" for modulo
26  - "^" for power
27  - For more advanced equasion use:
28  - "sqrt" for square root
29  - "sin" for sine calculations
30  - "cos" for cosine calculations
31  - "tan" for tangent calculations
32  - "sinh" for hyperbolic sine calculations
33  - "cosh" for hyperbolic cosine calculations
34  - "!" for factorial
35  - "|" to convert from one base to another (TODO)
36  - "C" to calculate choose notation
37  - "P" to calculate permutations
