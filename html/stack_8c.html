int stack_isEmpty ( stack *  s)

Check if the stack is empty.

The function will return a boolean based of a pointer to a stack It will return true (1) if the stack is empty and false (0) if the stack is populated. This function is important because performing operations on an empty stack can lead to errors due to garbage values being returned

Parameters
sthe pointer to the stack we check
Returns
boolean, true if stack is empty, false otherwise

int stack_isFull ( stack *  s)

Check if stack is full.

The function will return a boolean based of a pointer to a stack It will return true (1) if the stack is full and false (0) if the stack is not full (either empty or populated). This function is important because pushing to a full stack can lead to loss of data, or cause segfaults. pushing to the queue performs the check automatically.

Parameters
sthe pointer to the stack we check
Returns
boolean, true if stack is full, false otherwise
