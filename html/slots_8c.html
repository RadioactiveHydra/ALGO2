Function Documentation

int analyseResults ( char *  values[WIDTH])

Given a string of values rolled, give the result.

The functions analizes the string and returns a score based on the input.
The score is calculated like so:
For every characters, if there is 2 or more of the same character, add to the score 15(number_of_occurences!)
Example:
12345 -> 0
'11234' -> 45
'11223' -> 90
'11123' -> 90

Parameters
valuesa row from the slot machine
Returns
int the result
